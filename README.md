# Climate-Integrated Infrastructure Vulnerability Assessment System

An AI-powered decision-support tool that evaluates infrastructure risk using structural parameters and climate conditions.

## Features

- Climate-aware risk prediction
- Infrastructure vulnerability scoring
- AI-based inspection prioritization
- Explainable feature importance
- React frontend + FastAPI backend

## Tech Stack

Frontend:
React

Backend:
FastAPI

Machine Learning:
Random Forest (Scikit-learn)

Climate Data:
Open-Meteo API

## Workflow

1. User inputs structural data and location
2. Weather data is fetched automatically
3. ML model predicts vulnerability probability
4. System recommends inspection priority

## Example Output

Risk Category: Critical  
Recommended Action: Immediate structural audit
